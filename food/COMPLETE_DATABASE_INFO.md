## 🚀 Использование:

База теперь **полноценная** для любого фитнес-трекера или приложения по питанию!

```javascript
// Импорт
import foodDatabase from './food-database-complete.js';

// Получить все каши
const porridges = foodDatabase['Крупы и каши'];

// Найти продукт
function findProduct(name) {
  for (const category of Object.values(foodDatabase)) {
    const found = category.find(p => 
      p.name.toLowerCase().includes(name.toLowerCase())
    );
    if (found) return found;
  }
  return null;
}

// Пример
const rice = findProduct('рис'); 
// { name: 'Рис белый вареный', protein: 2.7, ... }
```
