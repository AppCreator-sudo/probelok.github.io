## 🔍 Поиск по сетям:

```javascript
function findByRestaurant(query) {
  const restaurants = [
    "McDonald's", "KFC", "Burger King", 
    "Subway", "Додо Пицца"
  ];
  
  return restaurants
    .filter(r => r.toLowerCase().includes(query.toLowerCase()))
    .map(r => ({ name: r, items: readyMeals[r] }));
}

// Пример
const mcMenu = findByRestaurant('mc'); 
// Вернет меню McDonald's
```
